## 🚀 Overview

This guide helps you set up **Stremio + Torrentio + VLC** for:

- ✅ Smooth streaming (no buffering)
- ✅ Better video quality
- ✅ Full control playback
- ✅ Clean UI experience

---

## 📥 Step 1: Install Stremio

1. Download Stremio from official website
2. Install normally
3. Launch the app

---

## 🔌 Step 2: Install Torrentio Add-on

1. Open **Add-ons**
2. Click **+ Add Addon**
3. Paste Torrentio link or configure via browser
4. Click **Install**

---

## ⚙️ Torrentio Recommended Settings

- Providers: Select common ones (YTS, 1337x, etc.)
- Sorting: **By quality then seeders**
- Exclude:
  - ❌ Cam
  - ❌ Screener
- Debrid Provider: **None (optional upgrade later)**

---

## 🎥 Step 3: Play Content

- Select any movie/series
- Choose:
  - Higher **seeders**
  - Reasonable **file size**
- Click and stream directly

👉 No need to download manually

---

## ⚠️ Problem: Buffering Issues

### ✅ Solutions:

- Choose smaller file (e.g. 2GB instead of 20GB)
- Pick higher seeders
- Avoid 4K on normal connection

---

## 🎬 Step 4: Install VLC (Recommended)

VLC gives:

- ✅ Better playback
- ✅ No UI bugs
- ✅ Full control
- ✅ Fixes aspect ratio issues

### 🔗 Download:
https://www.videolan.org/vlc/

---

## ⚙️ VLC Setup

1. Install VLC (default settings)
2. Open VLC once

---

## 🔗 Connect VLC with Stremio

1. Go to **Stremio Settings → Player**
2. Enable:

```
Play in external player → VLC
```

👉 Now videos will open in VLC automatically

---

## 🖥️ Fix Video Cropping (Black bars / cut video)

### ❌ Problem:
Video is cut from top & bottom

### ✅ Solution:

Press `Z`

✔ Cycles zoom modes  
✔ Restores full frame  

---

## 🎛️ VLC Controls (Useful)

- `F` → Fullscreen  
- `Z` → Zoom modes  
- `A` → Aspect ratio  
- `Space` → Pause/Play  

---
